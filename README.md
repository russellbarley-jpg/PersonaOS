# PersonaOS Setup Wizard

A self-service setup wizard that generates a personalised AI assistant for your Google Workspace. In about 10 minutes you get a script that sends you a daily morning briefing, lunchtime inbox triage, and end-of-day digest — all written in your own voice.

## What it does

Walk through 7 steps to configure:
- Which daily emails you want (morning briefing, lunchtime digest, EOD digest, tasks summary)
- Your personal and team details
- Your strategic priorities
- Your communication style (analysed from your sent emails via Claude)

At the end you receive a complete Google Apps Script file personalised to you. Paste it into a new Apps Script project and you're live.

## What you need before starting

- A Google account with Gmail, Calendar, and Tasks
- An Anthropic API key (request from IT Support AI Tools — select "Claude API only")
- About 10 minutes

## How to use

Open the app and follow the on-screen steps. No account or login required. Nothing you enter is sent to or stored on any server — all processing happens in your browser.
